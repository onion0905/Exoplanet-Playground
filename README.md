# Requirement
- Node.js
- python
# How to run
use a unix-like operation
1. clone the repo
2. setup the backend
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 backend/app.py
```
3. setup the frontend
```bash
cd frontend
npm i
npm run dev
```
4. setup the backend
```bash
python backend/app.py
```
5. Access the website through `http://localhost:3000`
